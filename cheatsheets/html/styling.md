# Styling

## Navbar

A full-width navbar that sticks to the top of the page when you scroll. 

```css
nav.navbar {
  position: fixed;
  top: 0;
  width: 100%;
}
```

See also [Bulma navbar](https://bulma.io/documentation/components/navbar/) documentation.

## Footer

```css
footer {
  position: fixed;
  bottom: 0;
  width: 100%;
}
```

Some padding may be useful too.
