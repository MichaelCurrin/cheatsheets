# Project packages

## Versions of package managers

- [Bundler versions](https://rubygems.org/gems/bundler/versions) on rubygems.org
- [pip](https://pypi.org/project/pip/) project on PyPI. See Release History.


## List installed packages

### Ruby

```sh
$ gem list

$ bundle list
```

### Python

```sh
$ pip list
$ pip freeze
```

### Node

```sh
$ npm list
$ npm -g list
```
