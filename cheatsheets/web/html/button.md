# Button

## Resources

- [Button tag](https://www.w3schools.com/tags/tag_button.asp) on W3 Schools.


## Basic

```html
<button>
    My text
</button>
```

When you hover over the button, the cursor will change to `pointer`. The same for `a` tags.


## Disabled

```html
<button disabled>
    My text
</button>
```
