# Font

From [W3 CSS Font](https://www.w3schools.com/css/css_font.asp) tutorial.

```css
p {
    font-family: "Times New Roman", Times, serif;
}

p.italic {
    font-style: italic;
}
```

```css
p {
    font-size: 40px;
}

p {
    font-size: 100%;
}

p {
    font-size: 2.5em;
}
```

```css

p.thick {
    font-weight: bold;
}
```
