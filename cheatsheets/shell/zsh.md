# ZSH

See ZSH [options](https://zsh.sourceforge.net/Doc/Release/Options.html).


## Navigation

```sh
DIRNAME

# Same as:
cd DIRNAME
```

```sh
..

# Same as:
cd ..
```


## Git

On install Git with Brew while using ZSH:


```
zsh completions and functions have been installed to:
  /usr/local/share/zsh/site-functions
```
