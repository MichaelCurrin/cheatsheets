# Commands


### Logging tip

From [loglevel](https://docs.npmjs.com/cli/v7/using-npm/config#loglevel) in config docs.

Change the verbosity of logging.

Default: `"notice"`

Values allowed:

> "silent", "error", "warn", "notice", "http", "timing", "info", "verbose", or "silly"


e.g.

```sh
$ npm install --loglevel warn
```
