# Git push cheatsheet

## Push branch

```sh
git push
```
```sh
git push origin master
```

## Push all branches
Push all branches. It's a good idea to deleted merged branches first to avoid recreating them on the repo.

```sh
git push --all
```


## Push tags only

NB. Does not push the branch itself.

```sh
git push --tags
```
