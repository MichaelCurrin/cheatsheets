# CLI cheatsheet

Cross-language guide for writing a command-line interface.

One argument

```
-d, --destination DESTINATION
```


One or more arguments.

```
--config CONFIG_FILE[,CONFIG_FILE2,...]
```

Short and long flags.

```
-d, --destination DESTINATION
```
