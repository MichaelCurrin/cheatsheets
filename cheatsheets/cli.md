# CLI cheatsheet

Cross-language guide for writing a command-line interface.


### One argument

```
-f, --foo FOO
```

### Optional argument

```
-f, --foo [FOO]   Your foo value.  (defaults to bar)
```


### One or more arguments

```
--foo FOO[,FOO2,...]
```

### Short and long flag

```
-f, --FOO FOO
```
